# Imbreezo - Cybersecurity Portfolio

## Overview
A personal portfolio website showcasing cybersecurity skills, projects, certifications, and contact information.

## Project Structure
- `maindex.html` - Main HTML page
- `codex.css` - Stylesheet
- `codex.js` - JavaScript functionality (particles animation, form handling)
- `server.py` - Simple Python HTTP server for development
- `mypfp.jpg` - Profile picture
- `lp.png` - Certification image

## Technology Stack
- Static HTML/CSS/JavaScript
- Particles.js for animated background
- Python HTTP server for local development

## Running Locally
The project runs via Python's built-in HTTP server on port 5000.

## Deployment
Configured as a static site deployment with the root directory as the public directory.
